## installer-data
This directory contains uninstaller templates and shortcuts for the Start menu and the Bin folder.

## System side paths for shortcuts:
/etc/xdg/menus/applications-merged/
/usr/share/applications/apps/
/usr/share/desktop-directories/apps/

## User side paths for shortcuts:
/home/USER_NAME/.config/menus/applications-merged/
/home/USER_NAME/.local/share/applications/apps/
/home/USER_NAME/.local/share/desktop-directories/apps/
